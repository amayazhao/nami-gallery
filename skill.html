核心能力

🧬
Two-Layer Architecture
Character Sheet（身份层）+ Scene Card（场景层）分离。身份特征跨场景锁定，场景随意切换不走形。
🎯
Pure Prompt Engineering
不依赖 LoRA / DreamBooth / 微调模型。纯 prompt + reference image 双重锚定，零训练成本。
🛡️
审核安全策略
积累了完整的审核绕过经验库。"不说身体说衣服" — 用服饰描述替代身体描述，安全且效果好。
📐
Prompt 经验库
眼神三要素、身材安全描述、成熟感调控公式、面料升级策略 — 系统化的 prompt engineering 知识沉淀。
🎭
Cosplay 模式
角色 Cos 其他角色时，保持核心身份特征（发色、眼色、发饰）不变，只换服装场景。
💰
极低成本
MiniMax image-01 仅 ¥0.025/张。40张一致性验证总成本 < ¥1.5。普通用户也能用得起。
