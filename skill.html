核心能力

🧬
Two-Layer 架构
Character Sheet（身份层）+ Scene Card（场景层）分离。身份特征跨场景锁定，场景随意切换不走形。
🔌
引擎可插拔
同一套代码支持 MiniMax 和腾讯混元 3.0。环境变量切换，上层零改动。未来可扩展更多引擎。
🎯
Pure Prompt
不依赖 LoRA / DreamBooth / 微调。纯 Prompt + 可选参考图，零训练成本，即开即用。
🛡️
审核安全
"不说身体说衣服" — 内置安全词库 + 自动降级重试。被拦截后自动调整 prompt 重试。
📐
身材分级
特写/半身/全身自动适配不同粒度的身体描述。宽松衣服自动补偿。构图决定描述深度。
🎭
Cosplay 模式
换装不换人。角色核心特征（发色、眼色、标志配饰）在 Cosplay 场景下仍然保持。
📄
YAML 配置化
角色定义完全在 character-sheet.yaml 中，gen.py 是稳定引擎。代码与配置分离。
💰
成本极低
MiniMax ¥0.025/张（40张/元）。混元 ¥0.20/张（5张/元）。零外部依赖*，纯 Python 标准库。
